# Test Strategy

## Test Pyramid
- Unit tests (70%) — Fast, isolated, single functions
- Integration tests (20%) — Component interactions
- E2E tests (10%) — Full user workflows

## Test Types
- Functional, Security, Performance, Accessibility, Property-based
